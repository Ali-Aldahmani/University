## 🚀 Core Topics Explored

This repository translates mathematical theory into practical Python code. Here are the core concepts covered:

* 🔹 **Vectors & Matrices:** Vector arithmetic, dot products, cross products, and matrix multiplication.
* 🔹 **Systems of Linear Equations:** Gaussian elimination, row reduction, and LU decomposition.
* 🔹 **Vector Spaces:** Subspaces, linear independence, basis, span, and dimension.
* 🔹 **Eigenvalues & Eigenvectors:** Characteristic equations, determinants, and diagonalization.
* 🔹 **Transformations:** Linear transformations, rotations, and projections.

---

## 💻 Tech Stack & Implementation

While the theory is purely mathematical, the execution relies heavily on programming. 

* **Language:** Python
* **Primary Library:** NumPy (used extensively for multidimensional arrays, matrices, and high-level mathematical functions).

All concepts are implemented from scratch or validated using NumPy's highly optimized linear algebra module (`numpy.linalg`).
